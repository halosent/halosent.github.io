# Live Orthodox

![Live Orthodox Banner](https://via.placeholder.com/1200x400)

A modern Orthodox Christian philosophy and apologetics platform dedicated to exploring **Eastern Orthodox Christianity**, theology, philosophy, Church history, and thoughtful responses to common objections.

**Owned and maintained by Unified.**

---

## ✨ About

**Live Orthodox** is a digital library and educational platform designed to help people explore Orthodox Christianity through:

* Philosophical discussions
* Biblical explanations
* Historical context
* The writings of the Church Fathers
* Responses to theological objections
* Analysis of alleged contradictions

The goal is to provide clear, respectful, and well-researched resources that encourage deeper understanding of Orthodox Christian thought.

---

## 🚀 Features

### 📚 Article Library

A searchable collection of Orthodox Christian articles covering:

* Theology
* Philosophy
* Apologetics
* Church history
* Biblical interpretation
* Ethics

Articles are written in a structured format with:

* Categories
* Tags
* Reading time
* References
* Related content

---

### 🔎 Contradictions Database

A dedicated section addressing commonly raised objections and alleged contradictions.

Each entry includes:

* The objection
* Context
* Orthodox response
* Biblical references
* Historical background
* Supporting sources

---

### 📖 Immersive Reader Mode

Every article includes a distraction-free reading experience.

Features:

* Fullscreen reading
* Adjustable font size
* Reading themes
* Line spacing controls
* Reading progress tracking
* Custom reader preferences

Designed for long-form theological and philosophical reading.

---

### 🎨 Modern Design

Built with a clean, elegant interface inspired by modern knowledge platforms.

Includes:

* Responsive design
* Smooth animations
* Animated backgrounds
* Orthodox-inspired visual elements
* Mobile-friendly layouts

## ✍️ Adding Articles

Articles are stored as Markdown files.

Example:

```md
---
title: "Example Article"
description: "An explanation of Orthodox theology."
author: "Unified"
date: "2026-01-01"
category: "Theology"
tags:
  - Orthodox
  - Philosophy
---

Article content goes here.
```

Adding new content does not require changing the website code.
